# Krystal Joy Brown — Portfolio

Static multi-page site: Home, Projects, Blog, Schoolwork.

## Structure

- `index.html`, `projects.html`, `blog.html`, `schoolwork.html` — top-level pages
- `projects/`, `blog/`, `schoolwork/` — detail page templates (copy an existing file in the folder to add a new entry, then link it from the matching index page)
- `css/style.css` — shared styles (colors, type, layout)
- `js/main.js` — nav, scroll effects

## Editing

Every page is plain HTML — open in any text editor, copy an existing `<a class="entry">` block on an index page (or duplicate a detail-page file) to add new content, then save.

## Hosting

Deployed via GitHub Pages from the `main` branch, root directory.
